# ssl-certs

Configuration for making ssl certificates with cfssl.

## Usage

The makefile will generate the following files:

* ca.pem
* ca.key
* intermediate-ca.pem
* intermediate-ca-key.pem
* beast-server.pem
* beast-server-key.pem
* beast-peer.pem
* beast-peer-key.pem
* beast-peer.pem
* beast-peer-key.pem
